# Craveiro e Sakarov Advogados Associados

Site institucional com múltiplas páginas para o escritório de advocacia Craveiro e Sakarov.

## Estrutura do Site

O site é composto por 4 páginas principais:

1. **index.html** - Página inicial com apresentação do escritório, destaques, resumo das áreas de atuação e processo de atendimento
2. **escritorio.html** - Página sobre o escritório, incluindo missão, visão, valores e diferenciais
3. **areas-atuacao.html** - Página detalhada com todas as áreas de atuação do escritório
4. **contato.html** - Página de contato com formulário, informações de contato e localização

## Características

- **Design Moderno**: Tema escuro com gradientes sutis e cores vibrantes (azul e verde)
- **Responsivo**: Adaptável a todos os tamanhos de tela (desktop, tablet e mobile)
- **Navegação Intuitiva**: Menu de navegação fixo com links para todas as páginas
- **Animações Suaves**: Transições e efeitos visuais profissionais
- **Formulário de Contato**: Sistema de validação de formulário
- **Botão WhatsApp**: Acesso direto para contato via WhatsApp
- **Scroll to Top**: Botão para retornar ao topo da página

## Áreas de Atuação

O site apresenta as seguintes áreas de atuação:

1. Direito Empresarial
2. Direito Tributário
3. Direito de Família
4. Direito Imobiliário
5. Direito do Trabalho
6. Direito Previdenciário
7. Registro de Marcas e Patentes

## Personalização

### Cores

As cores principais podem ser alteradas no arquivo `css/styles.css` nas variáveis CSS:

```css
:root {
    --primary-dark: #0a1118;
    --secondary-dark: #1a242f;
    --accent-blue: #00bcd4;
    --accent-green: #4caf50;
    --text-light: #e0e0e0;
    --text-muted: #a0a0a0;
    --white: #ffffff;
}
```

### Contato

Para alterar as informações de contato, edite os arquivos HTML nas seções de contato e rodapé.

### WhatsApp

Para alterar o número do WhatsApp, substitua `5561999999999` pelo número desejado no formato internacional nos links:

```html
<a href="https://wa.me/5561999999999" target="_blank" class="btn-whatsapp">Whatsapp</a>
```

## Como Usar

1. Abra o arquivo `index.html` em um navegador web
2. Navegue pelas páginas usando o menu de navegação
3. O site é totalmente funcional e pode ser hospedado em qualquer servidor web
